# NextGenScadaProgrammer Portfolio Site

Welcome to the source code for my professional portfolio site!  
Built to showcase my Industrial IoT, SCADA, and Automation projects with a focus on clean design, mobile optimization, and modular code architecture.

---

## 📂 Project Structure

/ ├── index.html # Home page ├── /projects/ # Project marketing pages │ └── /modbus-mqtt-sparkplugb/ │ └── index.html # Modbus to MQTT SparkplugB Gateway page ├── /blogs/ # Blog listing and posts │ ├── index.html │ └── blog-0.html ├── /partials/ # Shared reusable components │ ├── head.html # Global <head> (favicon, Tailwind, meta) │ ├── header.html # Top navigation bar │ └── footer.html # Footer with social links ├── /public/ │ ├── /assets/ # Public static assets (images, diagrams) │ │ └── architecture_diagram.png │ ├── /css/ │ │ └── style.css # Global custom styles (scrollbars, tweaks) │ └── /js/ │ └── main.js # Global JavaScript (head injection, nav highlighting, AOS init)


---

## 🛠 Technologies Used

- **HTML5** + **Tailwind CSS**
- **JavaScript (Vanilla)** for modular dynamic loading
- **AOS (Animate on Scroll)** for smooth animations
- **Vercel** for fast, scalable deployment

---

## 🧹 Key Features

- Fully **mobile-first** responsive design
- Dark theme with custom dark scrollbars
- Modular structure with dynamic header, footer, and head injection
- Automatic page title setting based on URL path
- Smooth hover animations on navigation and footer icons
- Clean project and blog section organization
- Easy expansion: just add new folders under `/projects/` or `/blogs/`

---

## 🚀 How to Add a New Project

1. Create a new folder inside `/projects/your-project-name/`
2. Add an `index.html` for your project marketing page
3. Add any needed images inside `/public/assets/`
4. Link to your new project page from the homepage or projects section
5. Deploy via GitHub → Vercel auto-publishes!
