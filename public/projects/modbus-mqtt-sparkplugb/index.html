Modbus to MQTT SparkplugB Gateway

This project is a Python-based industrial gateway designed to bridge legacy Modbus devices into modern IIoT systems using MQTT SparkplugB protocol. Features include configurable web UI, automatic buffering during connection loss, deadband support, and flexible Modbus RTU/TCP input options.

Deployed in rugged remote environments with cellular and satellite backhauls. Optimized for minimal bandwidth usage and scalable architecture.
