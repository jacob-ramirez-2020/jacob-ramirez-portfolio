Modbus to MQTT SparkplugB Gateway

A Python-based Industrial IoT gateway application that reads data from Modbus devices (both RTU and TCP) and publishes real-time data using the SparkplugB MQTT protocol. Designed for remote deployments, it provides deadband filtering, automatic buffering on disconnection, and easy configuration management.

View Full Project on GitHub

Key Features

  • ✅ Supports both Modbus RTU (RS485) and Modbus TCP (Ethernet/IP)
  • ✅ Publishes SparkplugB-compliant MQTT payloads
  • ✅ Deadband filtering to optimize network traffic
  • ✅ Local buffering using SQLite database for store-and-forward
  • ✅ Configurable via planned web-based UI (coming soon)
  • ✅ Built-in diagnostics and runtime status monitoring
  • ✅ Future plans for Docker deployment and dynamic configuration APIs
